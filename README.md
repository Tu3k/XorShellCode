# XorShellCode
Xor Encry ShellCode With x86

以命令行传参生成加密文件的方式，生成方式/框架大同小异，只做记录

### 使用（22/5/4优化）:

1. CS生成x86 payload
```
XorEncry payload.c
```
当前目录下生成payload.txt

2. 将加密代码放到shellcode.sln项目中，生成即可。
